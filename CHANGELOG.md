# 📜 CHANGELOG

Historial de versiones, mejoras y cambios de `research-frameworks`.

## [1.0.0] - 2026-04-01

### 🎉 Release Inicial

#### Added
- ✨ Skill core: investigación sistemática de frameworks
- 📊 Matriz de evaluación con 7 criterios estándar
- 🌳 Generador de árboles de decisión contextuales
- 🔗 Integración multi-fuente: GitHub, Reddit, blogs técnicos
- 📈 Scoring normalizado y ponderado
- 📝 Documentación completa (SKILL.md, TRIGGERS.md)
- 🔍 TRIGGERS.md con patrones de activación
- 💾 Estructura JSON para matrices comparativas
- 🎓 Ejemplos de casos de uso reales
- 📚 Quick start guide para nuevos usuarios
- 🛣️ Roadmap de adopción (3-6 semanas)
- 📞 Soporte y guía de preguntas frecuentes

#### Features
- Soporte para cualquier dominio técnico (backend, frontend, DevOps, IA, etc)
- Personalización de criterios de evaluación
- 3 niveles de profundidad (LIGHT, STANDARD, DEEP)
- Filtros por restricción (presupuesto, timeline, compatibility)
- Generación automática de ejemplos de código
- Detección de contexto en conversación

#### Documentation
- SKILL.md: 800+ líneas, guía completa
- TRIGGERS.md: 450+ líneas, patrones de activación
- FRAMEWORKS.json: estructura normalizada para datos
- examples/quick_start.md: primeros pasos
- Arquitectura clara y reproducible

#### Testing
- ✅ Verificado con casos reales (backend frameworks, frontend libs)
- ✅ Evaluado contra mejores prácticas (SOLID, modularidad)
- ✅ Validado con contexto de Wei (Bamboo, BAMI, GWOP)

#### Known Limitations
- Requiere al menos 3 opciones para comparar efectivamente
- Investigación basada en datos disponibles hasta enero 2025
- No incluye análisis de costos muy específicos (uso basado en horas)
